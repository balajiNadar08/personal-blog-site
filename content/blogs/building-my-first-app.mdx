# Why I Started

I've always wanted a place to share things I'm learning. Social media posts disappear quickly, but blogs remain useful for years.

## Tech Stack

I'm intentionally keeping things simple:

- Next.js
- TypeScript
- Tailwind CSS
- MDX
- Buttondown

No CMS. No database. Just markdown files.
