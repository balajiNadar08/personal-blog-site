# Looking Back

When I started, I thought programming was mostly about syntax.

I was wrong.

## Things I Believed

- I needed to know everything.
- I had to memorize syntax.
- Good developers never use Google.

## Things I Learned

- Reading documentation is a superpower.
- Debugging is half the job.
- Building projects teaches more than tutorials.

## My Favorite Quote

> Programming isn't about what you know; it's about what you can figure out.

## Next Goals

- Learn backend development.
- Write more.
- Contribute to open source.
